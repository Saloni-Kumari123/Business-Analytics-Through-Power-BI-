Business Analytics Through Power BI
Project Overview
This project demonstrates how Power BI can be used for business analytics.
It provides interactive dashboards to analyze marketing and sales performance data.

Key Insights:
Social media engagement (Facebook, Instagram, TikTok, etc.)
Organic traffic, CTR, exit rate, backlinks
Conversion rates by month, landing pages, and regions
Performance by media type (Video, Shorts, Infographics, Product pages)

How to Use:
Open the .pbix file in Power BI Desktop.
Explore the interactive dashboard with filters and slicers.
Publish to Power BI Service for sharing and collaboration.

Tools & Technologies:
Power BI Desktop
Power Query (data cleaning)
DAX (data modeling and calculations)
